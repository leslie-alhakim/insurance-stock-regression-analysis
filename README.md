# Insurance Stock Regression Analysis

A predictive analytics project exploring financial variables associated with stock prices in the **S&P 1500 Insurance Industry** using regression modeling in **R**.

## Project Overview

This study investigates whether financial indicators can predict stock price in insurance companies.

Variables analyzed:

- Earnings Per Share (EPS)
- Book Value Per Share (BVPS)
- Current Ratio (CR)
- Debt-to-Asset Ratio (DTA)

Methods included both:

- Multiple Linear Regression
- Nonlinear / Nonparametric Regression (GAM)

## Key Findings

📌 **BVPS had the strongest positive relationship with stock price**

📌 The regression model explained **63.77% of price variation (R² = 0.6377)**

📌 **EPS, BVPS, and CR were statistically significant predictors**

📌 The overall model was statistically significant.
