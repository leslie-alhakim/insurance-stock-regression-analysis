# Insurance Stock Regression Analysis
Analysis of financial indicators predicting stock prices in the insurance industry using regression modeling and exploratory data analysis in R.

## Key Analysis

This project analyzed financial variables associated with stock prices in the insurance sector:

- Earnings Per Share (EPS)
- Book Value Per Share (BVPS)
- Current Ratio (CR)
- Debt-to-Asset Ratio (DTA)

Methods included:
- Histograms
- Scatterplots
- Correlation analysis
- Linear regression
- Nonparametric regression
